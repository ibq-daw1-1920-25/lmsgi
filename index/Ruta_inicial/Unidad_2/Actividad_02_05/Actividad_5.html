Herramientas de desarrollo web

Tabla comparativa sobre las herramientas de desarrollo web
 CARACTERÍSTICAS PRINCIPALES   VENTAJAS   DESVENTAJAS 
 APTANA   Basado en elipse. 
 Asistente de código para HTML y Javascript. 
 Herramientas para trabajo con base de datos. 
 Multiplataforma.  		 Permite comprobar la compatibilidad de las funciones con los diferentes navegadores. 
 Sincronización con carpetas locales y remotas. 
 Incluye plugins para Eclipse.  		 Consumo de recursos. 
 ATOM   Desarrollado por miembros de la plataforma GitHub y esto hará que nuestro proyecto se sincronice automàticamente con el repositorio de Git. 
  La mayor parte de los paquetes tienen licencias de software libre. 
 Los documentos son totalmente editables y ajustables a nuestras preferencias.  		 Rendimiento lento. 
 BRACKETS   Editor de código creado por Adobe, Open Source. 
 Enfocado principalmente para desarrollo web. 
 Software gratuito y de código abierto. 
  Gratuito. 
 Multiplataforma. 
 Ademas de HTML, CSS y JavaScript, ofrece soporte para PHP, C, C++, Java, Python. 
 Posibilidad de modificar en tiempo real cualquier parte del código y ver los cambios. 
  Sólo trabaja con UTF-8.   Se asocia con con archivos de extensión PHP. 
 NOTEPAD++   Editor gratuito de código fuente libre que sustituye al Notepad. 
 Funciona en entorno MS Windows. 
 Su uso está regulado por la Licencia Pública General de GNU.  		 Aunque está diseñado especialmente para Microsoft, se puede instalar en otro SO. 
 Apertura de archivos en pestañas totalmente organizables con vistas alternativas para comparar texto. 
  No tiene tantas funcionalidades a la hora de hacer grandes textos. 
 SUBLIME   Editor de texto pensado para cualquier lenguaje de programación. 
 Multiplataforma. 
  Programa rápido en ejecución. 
 Gran cantidad de paquetes. 
 Ligero, apenas ocupa espacio por lo que consume pocos recursos.  		 No es gratuito. 
 Se necesita licencia para poder utilizar la versión más avanzada. 
 VISUAL STUDIO CODE   Editor de texto gratuito y de código abierto. 
 Desarrollado por Microsoft. 
 Multiplataforma.  		 Compatible con varios lenguajes de programación. 
 Interfaz simple. 
 Característica de la integración Git. 
 Permite una gran personalización. 
  Debes incorporar muchas de las funcionalidades a través de plugins. 
 Incomodidad a la hora de cambiar la configuración.
